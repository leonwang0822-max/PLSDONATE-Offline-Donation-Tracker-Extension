# PLS DONATE Offline Donation Tracker Extension

Real-time companion for the Offline Donation Tracker. View live donations and get notifications directly in your browser.

## Installation Instructions

1. **Download the Extension**
   - Click on the green **Code** button above and select **Download ZIP**.
   - Extract the ZIP file to a folder on your computer.

2. **Open Chrome Extensions Page**
   - Open Google Chrome.
   - In the address bar, type `chrome://extensions` and press Enter.

3. **Enable Developer Mode**
   - In the top right corner of the Extensions page, toggle the switch for **Developer mode** to **ON**.

4. **Load the Extension**
   - Click the **Load unpacked** button that appeared in the top left.
   - Select the folder where you extracted the extension files (or this repository folder).
   - The extension "PD Donation Tracker" should now appear in your list.

## Usage

1. **Pin the Extension**: Click the puzzle piece icon in Chrome and pin "PD Donation Tracker".
2. **Log In**: Click the extension icon and click "Open Dashboard" to log in to your account.
3. **Sync**: Once logged in, the extension will automatically sync and start showing your latest donations.
